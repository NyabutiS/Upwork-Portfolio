# Retail Sales Star Schema & Analytics

## Business Context
A retail chain needed a data warehouse design for sales analytics.

## Problem Statement
- Data scattered across OLTP systems.
- Reports were slow and inconsistent.
- Decision-makers lacked clear insights.

## Solution
- Designed **Star Schema**: FactSales + DimCustomer, DimProduct, DimDate.
- Sample **ETL scripts** for loading data.
- SQL queries for KPI reporting.
- Power BI dashboard for visualization.

## Deliverables
- Star Schema Diagram: `star_schema.png`
- ETL Sample Scripts: `etl_samples.sql`
- KPI Queries: `report_queries.sql`
- Power BI Dashboard: `sales_dashboard.pbix`
- Dashboard Screenshots: `screenshots/dashboard1.png`, `dashboard2.png`

## Tools Used
- SQL Server 2022
- Power BI
- dbdiagram.io

## Business Impact
- Fast, reliable reporting.
- Executive insights and trend analysis.
- Ready foundation for advanced analytics.
